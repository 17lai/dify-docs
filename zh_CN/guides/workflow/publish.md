# 应用发布

调试完成之后点击右上角的「发布」可以将该工作流保存并快速发布成为不同类型的应用。对话型应用支持发布为：

* 直接运行
* 嵌入网站
* 访问 API

工作流应用支持发布为：

* 直接运行
* 批处理
* 访问 API
