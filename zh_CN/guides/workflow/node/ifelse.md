# 条件分支

条件分支节点允许你根据 if/else 条件将 workflow 拆分成两个分支。在条件分支节点，你可以设置一个或多个 IF 触发条件，满足 IF 条件判断时触发 「IS TRUE」 下一步流程，未满足 IF 条件判断时触发「IS FALSE」下一步流程。
