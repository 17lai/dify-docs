# 工具

工作流内提供 Dify 的内置工具或自定义工具，在使用工具之前，你需要先给工具进行「授权」。若内置工具无法满足使用需求，你可以在「Dify—工具」内先创建自定义工具。

配置工具节点一般分为两个步骤：

1. 对工具授权/创建自定义工具
2. 配置工具输入和参数

如何创建自定义工具和配置工具请参考[工具配置说明](https://docs.dify.ai/v/zh-hans/guides/tools)。
